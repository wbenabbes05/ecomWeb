🛒 EcomWeb

A modern and responsive e-commerce web application built with React and Tailwind CSS. The project provides a clean shopping experience with product browsing, category filtering, and cart management while following modern frontend development practices.

🚀 Features
Responsive design for desktop, tablet, and mobile devices
Modern and intuitive user interface
Product listing and product details pages
Category-based product filtering
Shopping cart functionality
Reusable React components
Fast and optimized performance with Vite
Clean and maintainable code structure
🛠️ Technologies Used
React
Tailwind CSS
JavaScript (ES6+)
Vite
React Router
